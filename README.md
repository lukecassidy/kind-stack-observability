# kind-stack-observability

A dev Kubernetes observability stack on top of a local **kind** cluster.

- **Prometheus** (metrics)
- **Grafana** (dashboards)
- **Elasticsearch + Fluent Bit + Kibana** (logs)
- **podinfo** sample app (HTTP, metrics, logs)

This is **only for local development and demos**:

---

## Prerequisites

- Docker
- kind
- kubectl
- helm
- helmfile
